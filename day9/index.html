HTML & CSS Basics Summary

HTML Summary

HTML (Hypertext Markup Language) is used to define your page content, structure and meaning. You don't use it for styling purposes. Use CSS instead !

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content and then a closing tag
  • You can also have voir (empty) elements like images
  • You can also configure elements with attributes
  • Theres a long list of available elements but you ll gain experience over time, no worries.

Learn more about all available HTML elements on MDN HTML element reference.
